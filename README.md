# Healthcare AI Studio

**Developed at Harvard University**

This software suite is the official companion to the Healthcare AI course taught by **Dr. Trishan Panch** and **Prof. Rifat Atun**.

It provides a "batteries-included" development environment optimized for Health Tech innovation, designed to help students and practitioners build scalable, clinically safe, and commercially viable AI applications.

**No additional setup required.** Just clone this repository and start building.
